Research Statement

My primary research goal is to build intelligent, human-centred systems that fuse multiple data modalities to improve pattern recognition and decision-making in safety-critical contexts. That includes two tightly linked strands: (1) multimodal sensing and adaptive models for reliable event detection (e.g., visual + physiological + audio), and (2) securing IoT and cyber-physical systems using machine learning approaches that are practical for constrained environments (edge and embedded devices).

On the cybersecurity side, I have developed and evaluated ML-based intrusion detection systems for IoT-powered smart grids, comparing feature-selection strategies (Information Gain, PCA, and a hybrid IG+PCA) and models such as Random Forest and XGBoost on the CICIoT-2023 benchmark. That work shows how hybrid selection can improve detection of rare or sophisticated attacks while balancing runtime and accuracy — a critical requirement for real-time CPS protection. :contentReference[oaicite:2]{index=2}

Complementing network security, my work on deep learning for behavioral security (for example, Bi-LSTM models for keylogger detection) demonstrates how sequence and temporal models can be applied to non-traditional telemetry (keystroke dynamics, host activity) to detect stealthy threats. Together, these efforts form a research agenda that spans sensing, multimodal fusion, interpretable ML, and deployable security mechanisms for the edge. :contentReference[oaicite:3]{index=3}

Current themes I am actively pursuing:

  • Multimodal fusion strategies (attention, cross-modal transformers, late/early fusion) for robust event detection in safety-critical settings.
  • ML-based IDS and anomaly detection for IoT & smart grid topologies — emphasizing practical feature selection, low-latency inference, and class-imbalance solutions.
  • Sequence and temporal models (Bi-LSTM, temporal CNNs) for host-level threat detection (e.g., keyloggers, process anomalies).
  • Interpretable, resource-efficient architectures that can run on edge/embedded devices and adapt as attack patterns evolve.

More details, reports, datasets and code are available in my project folder: AI + Cybersecurity — project folder.
